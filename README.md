# 📊 Análisis de Ventas con IA (Groq)

Proyecto de **Análisis Exploratorio de Datos (EDA)** sobre un conjunto de datos de ventas utilizando **Python, Pandas y modelos de IA generativa** a través de Groq.

El objetivo es combinar **análisis de datos tradicional** con **modelos de lenguaje (LLMs)** para obtener **insights de negocio de forma automática**.

---

# 🎯 Problema de negocio

Las empresas necesitan comprender sus **patrones de venta** para mejorar sus estrategias comerciales, optimizar promociones y ajustar la oferta de productos.

Este proyecto analiza un dataset de ventas para responder preguntas como:

- ¿Qué categorías de productos generan más ventas?
- ¿Qué localidades presentan mayor actividad comercial?
- ¿Las promociones impactan en las ventas?
- ¿Existen oportunidades comerciales en ciertas categorías?

El objetivo es transformar datos en **información útil para la toma de decisiones**.

---

# 🚀 Descripción del Proyecto

Este proyecto analiza un dataset de ventas para descubrir **patrones, tendencias y oportunidades comerciales**.

El flujo de trabajo incluye:

- Carga de datos
- Limpieza y preparación del dataset
- Análisis exploratorio de datos (EDA)
- Generación de estadísticas y métricas
- Interpretación automática de resultados mediante IA

---

# 🤖 Uso de IA en el análisis

Para complementar el análisis exploratorio, se utilizó un **modelo de lenguaje (LLM)** mediante la API de Groq.

El modelo recibe un resumen del dataset y genera **interpretaciones automáticas del comportamiento de las ventas**, produciendo insights de negocio que ayudan a entender los datos desde una perspectiva comercial.

---

# 📑 Informe de negocio generado por LLM

## 📊 Patrones de ventas

- **Mayor cantidad de ventas en la categoría "Fiesta"**  
  La categoría de *Fiesta* representa aproximadamente el **62,5% de las ventas totales**, lo que indica una alta demanda de productos relacionados con eventos y celebraciones.

- **Menor cantidad de ventas en la categoría "Oficina"**  
  Esta categoría representa alrededor del **15,9% de las ventas**, lo que sugiere menor demanda comparada con otras categorías.

- **Mayor volumen de ventas en la localidad de Buenos Aires**  
  Buenos Aires concentra aproximadamente el **42,3% de las ventas**, lo que indica una fuerte actividad comercial en esta región.

---

## 🛍 Productos fuertes

- **La categoría "Fiesta" presenta el mayor volumen de ventas**, destacándose como la categoría con mejor desempeño dentro del dataset.

- Los productos relacionados con **decoración, entretenimiento y celebraciones** parecen tener una mayor demanda dentro del mercado analizado.

---

## 💼 Recomendaciones comerciales

- **Incrementar la oferta de productos en la categoría "Oficina"**  
  Aunque presenta menor demanda, podría explorarse el crecimiento de esta categoría mediante nuevas estrategias comerciales.

- **Crear campañas de marketing enfocadas en Buenos Aires**  
  Dado el alto volumen de ventas en esta localidad, se pueden diseñar campañas específicas para potenciar aún más el mercado.

- **Analizar la competencia en la categoría "Fiesta"**  
  Debido a su alta demanda, es importante estudiar el mercado para identificar oportunidades de diferenciación.

---

# 📊 Conclusiones del análisis

A partir del análisis exploratorio y el informe generado por IA se identificaron los siguientes puntos clave:

- La categoría **Fiesta domina el volumen de ventas** dentro del dataset.
- **Buenos Aires aparece como el mercado más activo** en términos de ventas.
- Existe potencial de crecimiento en categorías con menor participación como **Oficina**.
- Las herramientas de **IA generativa pueden complementar el análisis de datos** generando interpretaciones automáticas de los resultados.
